
# Implementacja algorytmu KMM (Karatsuba Matrix Multiplication) w Verilogu

Projekt przedstawia sprzętową implementację szybkiego algorytmu mnożenia macierzy z wykorzystaniem metody Karacuby (KMM - Karatsuba Matrix Multiplication) napisaną w języku Verilog w ramach przedmiotu OiAK (Organizacja i Architektura Komputerów).
